This is a project based on The Odin Project - Foundations section for JavaScript. 

It generates a 16x16 grid at the beginning, generates random colors on hover on each of the cells, and they get darkened by 10% on each hover, making them black by 10 hovers per cell. 
It has 2 buttons, one to create a grid of the size that the user wants (with a limit of 100x100 for performance reasons), and another one to clear the grid. 
The project is made to practice for DOM manipulation in JS.
